# 🪶 Obsidian Proofread Plugin for Claude Code
### Claudeで音声入力原稿を一発校正するプラグイン

---

## 📘 概要

このプラグインは、Obsidian 上で編集中のノートを  
**Claude Code（CLI版）** に渡して自動で校正するためのツールです。  
音声入力で作成した台本や、話し言葉が混ざった原稿を  
自然な書き言葉に整えます。

---

## 🚀 使い方

1. Obsidian を開いて、Markdown ノートを編集します。  
2. 右上のヘッダーに「校正」ボタンが追加されます。  
3. ボタンをクリックすると Claude Code が起動し、  
   ノート全文を校正します。  
4. 数秒後、校正後の文章がエディタに自動で反映されます。

> 💡 校正中は「校正中…」と表示されます。  
> 処理が完了すると自動的に元の表示に戻ります。

---

## 💬 校正内容の方針

- 誤字脱字・誤変換の修正  
- 口語的な表現を自然な書き言葉に整える  
- 助詞の誤用や句読点の不足を補う  
- 意味や文体を変えず、最小限の修正にとどめる  

---

## 🧪 テスト例

音声入力で作成した以下のような文章を貼り付けて試すことができます。

> えっと今日はAIの活用についてしゃべっていこうとおもいます。  
> 最近ほんとにAIが色んなとこでつかわれてて、たとえば仕事の効率化とか、  
> あとはイラスト作成とか、ほんとに幅広くなってます。

---

## ⚙️ 前提条件

- Claude Code CLI がインストール済みであること  
- Node.js v20 以上を推奨  
- Obsidian コミュニティプラグインを有効化済み  

---

## 🧩 特徴

- Obsidian から直接 Claude Code を呼び出して動作  
- 校正処理はローカルで完結（インターネット不要）  
- Claude Sonnet 4.5 モデル対応  
- ファイル権限の許可不要（テキスト直渡し方式）
